This project uses the last.fm API. I did a couple of things with the API. The first thing I did was create an artist search. The search fetches the artit's last.fm summary, play and follower count. This API has many features, I chose to display the current top tracks playing on last.fm, look below! And finally I created a code that generates a user's top artists of all time. The last bit only works if you have a last.fm account.
